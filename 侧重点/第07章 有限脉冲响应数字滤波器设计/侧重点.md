# 第07章 有限脉冲响应数字滤波器设计 — 侧重点

## 第07章 有限脉冲响应数字滤波器设计

### 薄弱点 1：窗函数法设计公式

**错误类型**：计算公式不熟练

**复习建议**：
1. 背住理想低通脉冲响应：$h_d(n) = \frac{\sin[\omega_c(n-\alpha)]}{\pi(n-\alpha)}$，$\alpha = (N-1)/2$
2. 记住特殊点：$n = \alpha$ 时用洛必达，$h_d(\alpha) = \omega_c/\pi$
3. 最终滤波器 $h(n) = h_d(n) \cdot w(n)$，就是理想响应乘以窗函数
4. 海明窗公式考试通常会给，不用硬背

### 薄弱点 2：吉布斯现象（简答背诵）

**复习建议**：背住三句话：
1. **是什么**：矩形窗截断导致频域边缘振荡和过冲
2. **特点**：增大 $N$ 振荡变窄但峰值不减（约9%）
3. **解决**：用旁瓣更小的窗函数代替矩形窗

### 薄弱点 3：IIR 与 FIR 对比（简答背诵）

**复习建议**：背住四个维度各一句话：
1. 稳定性：IIR 不一定稳定，FIR 天然稳定
2. 线性相位：IIR 做不到，FIR 可以
3. 阶数：IIR 低，FIR 高
4. 设计方法：IIR 间接（借模拟），FIR 直接（窗函数）
