# 第05章 时域离散系统的网络结构 — 侧重点

## 第05章 时域离散系统的网络结构

### 薄弱点 1：FIR 三种结构记混

**错误类型**：IIR 和 FIR 的结构类型混淆

**具体表现**：不知道 FIR 的第三种结构是"线性相位型"，猜成了"并联型"

**复习建议**：
1. IIR 三兄弟：直接型、级联型、**并联型**
2. FIR 三兄弟：直接型、级联型、**线性相位型**
3. 口诀：IIR 有"并联"（因为可以做部分分式展开），FIR 有"线性相位"（因为 $h(n)$ 可以对称）
4. 线性相位是 FIR 独有的优势，IIR 做不到

### 薄弱点 2：反馈系数来自分母还是分子搞混

**错误类型**：H(z) 分子分母与结构图左右侧的对应关系记反

**具体表现**：把"反馈来自分母"答成了"反馈来自分子"

**复习建议**：
1. 死记：**左反馈 = 分母取反，右前馈 = 分子原样**
2. 推导理解：$H(z)$ 的分母对应差分方程的 $y(n-k)$ 项（左边），移到右边就变号 → 所以反馈系数 = 分母系数取反
3. 考试画直接型时，第一步就是把 $H(z)$ 的分母系数全部变号列出来作为反馈系数

### 薄弱点 3：级联型的优点不了解

**错误类型**：概念盲区

**复习建议**：
1. 级联型优点 = **独立控制零极点 + 系数灵敏度低**
2. 直觉：直接型的高阶多项式"牵一发动全身"，级联型拆开后"各管各的"
3. 这个知识点选择题经常考，属于送分题，必须拿到
