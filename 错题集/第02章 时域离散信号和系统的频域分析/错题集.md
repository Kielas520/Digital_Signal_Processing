## 第02章 时域离散信号和系统的频域分析

### 错题 1（2026-06-21）

**题目**：Z 变换的移位性质：x(n-m) 的 Z 变换为 ______。

**用户答案**：求和(z^{-m}X(z))

**正确答案**：z^{-m} X(z)

**错因**：将移位性质与 Z 变换的定义公式混淆。移位性质是简单的乘法关系 x(n-m) ↔ z^{-m}X(z)，没有求和操作。求和只出现在 Z 变换的定义 X(z)=Σx(n)z^{-n} 中。

### 错题 2（2026-06-21）

**题目**：$X(z) = \frac{z^2}{(z-2)(z-0.5)}$，$|z|>2$，用部分分式展开法求 $x(n)$。

**用户答案**：直接展开 X(z)（不除以 z），得 A=8/3, B=-1/6，但无法完成逆变换

**正确答案**：

第1步：将 $X(z)$ 除以 $z$

$$\frac{X(z)}{z} = \frac{z}{(z-2)(z-0.5)}$$

第2步：部分分式展开

$$\frac{z}{(z-2)(z-0.5)} = \frac{A_1}{z-2} + \frac{A_2}{z-0.5}$$

第3步：求系数

$$A_1 = (z-2) \cdot \frac{z}{(z-2)(z-0.5)}\bigg|_{z=2} = \frac{2}{2-0.5} = \frac{2}{1.5} = \frac{4}{3}$$

$$A_2 = (z-0.5) \cdot \frac{z}{(z-2)(z-0.5)}\bigg|_{z=0.5} = \frac{0.5}{0.5-2} = \frac{0.5}{-1.5} = -\frac{1}{3}$$

第4步：乘回 $z$

$$X(z) = \frac{4}{3} \cdot \frac{z}{z-2} + \left(-\frac{1}{3}\right) \cdot \frac{z}{z-0.5}$$

第5步：收敛域 $|z|>2$，两极点均在收敛域内侧，因果序列，查表 $\frac{z}{z-a} \leftrightarrow a^n u(n)$：

$$x(n) = \frac{4}{3} \cdot 2^n u(n) - \frac{1}{3} \cdot (0.5)^n u(n)$$

验证：$x(0) = \frac{4}{3} - \frac{1}{3} = 1$；由初值定理 $x(0) = \lim_{z \to \infty}X(z) = 1$ ✓

**错因**：
1. 未掌握"先除以 z 再展开"的标准方法，导致展开后的形式为 $\frac{A}{z-p}$（分子无 $z$），无法直接查表
2. 标准流程是：X(z)/z → 展开 → 求系数 → 乘回 z → 得到 $\frac{z}{z-p}$ 的形式 → 直接查表

**章节**：第02章 — 逆Z变换（部分分式展开法）

**日期**：2026-06-21

### 错题 3（2026-06-21）

**题目**：$y(n) = 0.5y(n-1) + x(n)$，$x(n) = (0.8)^n u(n)$，$y(-1) = 2$，求 $y(n)$。

**用户答案**：称会写但未完成，要求给答案（视为答错）

**正确答案**：

第1步：对差分方程两边取单边 Z 变换

$$Y(z) = 0.5[z^{-1}Y(z) + y(-1)] + X(z) = 0.5z^{-1}Y(z) + 0.5 \times 2 + \frac{1}{1-0.8z^{-1}}$$

$$Y(z)(1-0.5z^{-1}) = 1 + \frac{1}{1-0.8z^{-1}}$$

$$Y(z) = \underbrace{\frac{1}{1-0.5z^{-1}}}_{\text{零输入（来自初始条件）}} + \underbrace{\frac{1}{(1-0.8z^{-1})(1-0.5z^{-1})}}_{\text{零状态（来自输入）}}$$

第2步：零输入响应（直接查表）

$$\frac{1}{1-0.5z^{-1}} \to (0.5)^n u(n)$$

第3步：零状态响应（部分分式展开）

$$\frac{1}{(1-0.8z^{-1})(1-0.5z^{-1})} = \frac{A}{1-0.8z^{-1}} + \frac{B}{1-0.5z^{-1}}$$

两边乘以分母：$1 = A(1-0.5z^{-1}) + B(1-0.8z^{-1})$

令 $z^{-1} = 1/0.8 = 1.25$：$1 = A(1-0.625) = 0.375A$，$A = \frac{8}{3}$

令 $z^{-1} = 1/0.5 = 2$：$1 = B(1-1.6) = -0.6B$，$B = -\frac{5}{3}$

逐项查表：$\frac{8}{3}(0.8)^n u(n) - \frac{5}{3}(0.5)^n u(n)$

第4步：合并

$$y(n) = \underbrace{(0.5)^n}_{\text{零输入}} + \underbrace{\frac{8}{3}(0.8)^n - \frac{5}{3}(0.5)^n}_{\text{零状态}}$$

合并 $(0.5)^n$ 项：$1 - \frac{5}{3} = -\frac{2}{3}$

$$y(n) = \frac{8}{3}(0.8)^n - \frac{2}{3}(0.5)^n, \quad n \geq 0$$

第5步：验证

$y(0) = \frac{8}{3} - \frac{2}{3} = 2$；原方程：$y(0) = 0.5 \times 2 + 1 = 2$ ✓

$y(1) = \frac{8}{3} \times 0.8 - \frac{2}{3} \times 0.5 = \frac{6.4-1}{3} = 1.8$；原方程：$y(1) = 0.5 \times 2 + 0.8 = 1.8$ ✓

**错因**：计算未完成（部分分式展开+逆变换的完整流程不够熟练）
