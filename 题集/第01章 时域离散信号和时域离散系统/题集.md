### 选择题 1（1分）

**题目**：已知 x(n) = {2, 5, 3, 7}（起始于 n=0），求 x(n)·δ(n-3) = ?

- A. 7
- B. 3
- C. 7δ(n-3) ✅
- D. 3δ(n-3)

**解析**：筛选性质：x(n)·δ(n-3) = x(3)·δ(n-3)。x(3)=7，故答案为 7δ(n-3)。注意结果是加权脉冲序列，不是常数。

### 选择题 2（1分）

**题目**：已知 x(n) = cos(0.5πn)，若为周期序列，则周期 N = ?

- A. 2
- B. 4 ✅
- C. 8
- D. 非周期序列

### 选择题 3（1分）

**题目**：对一个最高频率 f₀ = 5kHz 的连续信号进行采样，要无失真恢复，采样频率 fs 至少为：

- A. 2.5kHz
- B. 5kHz
- C. 10kHz ✅
- D. 20kHz

**解析**：采样定理 fs ≥ 2f₀ = 2 × 5kHz = 10kHz。

### 填空题 4（1分）

**题目**：LTI 系统具有因果性的充要条件是：h(n) = 0，当 ______ 时。

**答案**：n < 0

**解析**：因果性要求系统不依赖未来输入，反映在脉冲响应上就是 n<0 时 h(n)=0。

### 填空题 5（1分）

**题目**：两个长度分别为 N₁=4 和 N₂=5 的有限长序列做线性卷积，结果序列的长度为？

**题目**：已知 x(n) = R₄(n) = {1,1,1,1}（n=0,1,2,3），h(n) = R₄(n) = {1,1,1,1}（n=0,1,2,3），用列表法计算 y(n) = x(n) * h(n)，写出完整序列值及长度。

**标准答案**：

列表法：
```
              1   1   1   1       <- h(n)
          x   1   1   1   1       <- x(n)
          -----------------
  h(n)x(0):  1   1   1   1
  h(n)x(1):      1   1   1   1
  h(n)x(2):          1   1   1   1
  h(n)x(3):              1   1   1   1
          -------------------------
  y(n):      1   2   3   4   3   2   1
```

y(n) = {1, 2, 3, 4, 3, 2, 1}，n = 0,1,...,6，长度 = N₁ + N₂ - 1 = 4 + 4 - 1 = 7。

### 大题 2(1)：因果性充要条件与证明（复习PPT第15页）

**题目**：写出 LTI 系统具有因果性的充要条件，并证明其充分性。

**标准答案**：

充要条件：h(n) = 0，n < 0。

充分性证明：若 h(n) = 0（n < 0），则在卷积和 y(n) = Σ x(m)h(n-m) 中，只有 n-m ≥ 0（即 m ≤ n）的项非零，因此 n₀ 时刻的输出 y(n₀) 只与 m ≤ n₀ 时刻的输入 x(m) 有关，系统为因果系统。

**用户答案**：h(n)=0, n<0（条件正确）；证明表述为"卷积和响应两个环节"（方向正确但过于笼统，考试会扣分）。

### 大题 2(2)：稳定性充要条件与证明（复习PPT第16页）

**题目**：写出 LTI 系统具有稳定性的充要条件，并证明其充分性。

**标准答案**：

充要条件：单位脉冲响应绝对可和，即 Σ|h(n)| < +∞。

充分性证明：若 Σ|h(n)| < ∞，对任意有界输入 |x(n)| ≤ M，有：
|y(n)| = |Σ h(m)x(n-m)| ≤ Σ|h(m)||x(n-m)| ≤ M·Σ|h(m)| < ∞
故输出有界，系统稳定。

**用户答案**："一定要有界，通过卷积反向证明输入有界为M"（充要条件表述错误：把结论"输出有界"当成了条件，正确条件应为"h(n)绝对可和"）。
