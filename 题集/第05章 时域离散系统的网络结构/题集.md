# 第05章 时域离散系统的网络结构 — 题集

## 2026-06-21 选择填空测试

### 填空题 1（2分，真题原型）

**题目**：FIR 数字滤波器的网络结构主要有 ______、______ 和 ______ 三种形式。

**答案**：直接型、级联型、线性相位型

**解析**：FIR 特有的三种结构。注意线性相位型是 FIR 独有的（IIR 无法精确实现线性相位），并联型是 IIR 的结构，不是 FIR 的。

---

### 选择题 2（1分）

**题目**：画直接 II 型结构时，延迟器链左侧的反馈系数来自 H(z) 的？

- A. 分子系数不变
- B. 分子系数取反
- C. 分母系数不变
- D. 分母系数取反 ✅

**解析**：$H(z)$ 的分母对应差分方程中 $y(n-k)$ 项。从分母移到等式右边时符号取反，所以图中的反馈系数是分母系数取反后的值。分子系数对应前馈（右侧），符号保持不变。

---

### 选择题 3（1分）

**题目**：与直接型相比，级联型结构的主要优点是？

- A. 运算量更少
- B. 不需要延迟器
- C. 可以独立控制每对零极点，系数灵敏度低 ✅
- D. 可以实现线性相位

**解析**：级联型将高阶系统拆成多个低阶子系统，每个子系统独立控制一对零极点。调整某个子系统的系数只影响对应的零极点，不会"牵一发动全身"——这就是"系数灵敏度低"。

---

### 选择题 4（1分）

**题目**：IIR 与 FIR 在网络结构上的根本区别是？

- A. IIR需要更多延迟器
- B. IIR有反馈支路（输出回路），FIR没有 ✅
- C. FIR不需要乘法器
- D. IIR不能用直接型实现

**解析**：IIR 有反馈（输出 y 回来参与计算），所以脉冲响应无限长；FIR 无反馈，脉冲响应有限长。

---

### 填空题 5（1分）

**题目**：线性相位 FIR 滤波器的脉冲响应 $h(n)$ 满足对称条件：$h(n) =$ ______。

**答案**：$\pm h(N-1-n)$

### 大题 1：由结构图写差分方程和 H(z)（复习PPT第101页课堂练习）

**题目结构图**：

![P101 课堂练习结构图](../知识脉络/第05章%20时域离散系统的网络结构/图片/P101_课堂练习.png)

**标准答案**：

**第1步：从结构图中读出系数**

观察图中的信号流图：
- 有 2 个延迟器 $z^{-1}$ → 二阶系统
- 反馈支路（从延迟器输出回到输入端）系数：第1个延迟器回来乘以 **$2$**，第2个延迟器回来乘以 **$-0.8$**
- 前馈支路（从输入和延迟器输出到最终输出）系数：直通为 **$1$**，第1个延迟器出来乘以 **$3$**

**第2步：写差分方程**

反馈系数在图中的值就是移项后的值（即 $-a_k$），所以：
- $-a_1 = 2 \implies a_1 = -2$
- $-a_2 = -0.8 \implies a_2 = 0.8$

前馈系数直接读：$b_0 = 1$，$b_1 = 3$

差分方程（$y(n) = \sum(-a_k)y(n-k) + \sum b_k x(n-k)$）：

$$\boxed{y(n) = 2y(n-1) - 0.8y(n-2) + x(n) + 3x(n-1)}$$

**第3步：写系统函数**

对差分方程取 Z 变换：

$$Y(z)(1 - 2z^{-1} + 0.8z^{-2}) = X(z)(1 + 3z^{-1})$$

$$\boxed{H(z) = \frac{Y(z)}{X(z)} = \frac{1 + 3z^{-1}}{1 - 2z^{-1} + 0.8z^{-2}}}$$

### 大题 2：由 $H(z)$ 画直接型结构（真题考法）

**题目**：已知 $H(z) = \frac{1 + 3z^{-1}}{1 - 2z^{-1} + 0.8z^{-2}}$，（1）写出差分方程；（2）画直接 II 型时需几个延迟器？反馈和前馈系数分别是多少？

**标准答案**：

**（1）写差分方程**

$H(z) = Y(z)/X(z)$，交叉相乘：

$$Y(z)(1 - 2z^{-1} + 0.8z^{-2}) = X(z)(1 + 3z^{-1})$$

取逆 Z 变换：

$$y(n) - 2y(n-1) + 0.8y(n-2) = x(n) + 3x(n-1)$$

整理：

$$\boxed{y(n) = 2y(n-1) - 0.8y(n-2) + x(n) + 3x(n-1)}$$

**（2）直接 II 型结构参数**

最高幂次为 $z^{-2}$（二阶系统），需要 **2 个延迟器**。

分母多项式：$1 - 2z^{-1} + 0.8z^{-2}$，系数为 $a_1 = -2$，$a_2 = 0.8$

反馈系数（分母系数取反，即 $-a_k$）：

$$\boxed{\text{反馈：} -a_1 = 2, \quad -a_2 = -0.8}$$

分子多项式：$1 + 3z^{-1}$，系数为 $b_0 = 1$，$b_1 = 3$

前馈系数（分子系数保持原样）：

$$\boxed{\text{前馈：} b_0 = 1, \quad b_1 = 3}$$

画图方法：2 个 $z^{-1}$ 竖着串联在中间，左边挂反馈系数 $2$ 和 $-0.8$，右边挂前馈系数 $1$ 和 $3$。具体画法对照 PPT 第 97 页的结构图格式。

**解析**：正对称（$+$）或反对称（$-$）。利用这个对称性，可以将对称位置的输入先相加/减再乘系数，从而节省约一半的乘法器。
