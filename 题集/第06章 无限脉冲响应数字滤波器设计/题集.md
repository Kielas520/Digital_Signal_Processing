### 大题 1：双线性变换法（复习PPT第133页）

**题目**：$H_a(s) = \frac{1}{s^2+3s+2}$，$T=2$，用双线性变换法求 $H(z)$。

**标准答案**：

第1步：确定替换关系。$T=2$，所以 $s = \frac{2}{T}\frac{1-z^{-1}}{1+z^{-1}} = \frac{1-z^{-1}}{1+z^{-1}}$

第2步：因式分解分母 $s^2+3s+2 = (s+1)(s+2)$，逐项替换：

$$s+1 = \frac{1-z^{-1}}{1+z^{-1}}+1 = \frac{1-z^{-1}+1+z^{-1}}{1+z^{-1}} = \frac{2}{1+z^{-1}}$$

$$s+2 = \frac{1-z^{-1}}{1+z^{-1}}+2 = \frac{1-z^{-1}+2+2z^{-1}}{1+z^{-1}} = \frac{3+z^{-1}}{1+z^{-1}}$$

第3步：代入 $H_a(s)$：

$$H(z) = \frac{1}{\frac{2}{1+z^{-1}} \cdot \frac{3+z^{-1}}{1+z^{-1}}} = \frac{(1+z^{-1})^2}{2(3+z^{-1})}$$

第4步：展开化简：

$$H(z) = \frac{1+2z^{-1}+z^{-2}}{6+2z^{-1}}$$

### 大题 2(1)：脉冲响应不变法（复习PPT第136页）

**题目**：$H_a(s) = \frac{1}{(s+1)(s+2)}$，$T=2$，用脉冲响应不变法求 $H(z)$。

**标准答案**：

第1步：部分分式展开。

$$\frac{1}{(s+1)(s+2)} = \frac{A_1}{s+1} + \frac{A_2}{s+2}$$

$$A_1 = \frac{1}{s+2}\bigg|_{s=-1} = \frac{1}{1} = 1, \quad A_2 = \frac{1}{s+1}\bigg|_{s=-2} = \frac{1}{-1} = -1$$

$$H_a(s) = \frac{1}{s+1} - \frac{1}{s+2}$$

第2步：逐项替换极点。极点 $s_1=-1$，$e^{s_1 T}=e^{-2}$；极点 $s_2=-2$，$e^{s_2 T}=e^{-4}$。

替换规则：$\frac{A_k}{s-s_k} \to \frac{A_k}{1-e^{s_k T}z^{-1}}$

注意这里 $H_a(s) = \frac{1}{s-(-1)} + \frac{-1}{s-(-2)}$，所以 $s_1=-1, s_2=-2$。

$$H(z) = \frac{1}{1-e^{-2}z^{-1}} - \frac{1}{1-e^{-4}z^{-1}}$$

### 大题 2(3)：两种方法优缺点对比

**题目**：简述脉冲响应不变法和双线性变换法各自的优缺点。

**标准答案**：

脉冲响应不变法的主要优点是模拟频率和数字频率之间的关系是线性的（$\omega = \Omega T$），时域逼近好；其主要缺点是存在频谱混叠。

双线性变换法的主要优点是避免了频谱混叠；其缺点是模拟频率和数字频率之间的关系是非线性的（$\Omega = \frac{2}{T}\tan\frac{\omega}{2}$），需要进行频率预畸变校正。
