# 零输入响应与零状态响应的判断

## 核心概念

解差分方程时，完整输出 = 零输入响应 + 零状态响应：

$$
y(n) = y_{zi}(n) + y_{zs}(n)
$$

| 名称 | 含义 | 怎么得到 |
|------|------|---------|
| **零输入响应** $y_{zi}(n)$ | 没有外部输入时，仅靠初始条件产生的输出 | 令 $x(n) = 0$，只保留初始条件 |
| **零状态响应** $y_{zs}(n)$ | 初始状态为零时，仅靠输入驱动产生的输出 | 令所有初始条件 $= 0$，只保留 $X(z)$ |

## 怎么从 $Y(z)$ 中判断哪项是哪个？

### 关键步骤

对差分方程取 Z 变换并整理后，$Y(z)$ 的表达式自然分成两部分：

$$
Y(z) = \underbrace{\frac{\text{含初始条件的项}}{1-bz^{-1}}}_{\text{零输入响应}} + \underbrace{\frac{X(z)}{1-bz^{-1}}}_{\text{零状态响应}}
$$

### 判断规则

**一句话：跟着初始条件走的是零输入，跟着 $X(z)$ 走的是零状态。**

- 含 $y(-1)$、$y(-2)$ 等初始条件，但**不含** $X(z)$ → **零输入响应**
- 含 $X(z)$，但**不含**初始条件 → **零状态响应**

## 以例题为例

**差分方程**：$y(n) = by(n-1) + x(n)$，$x(n) = a^n u(n)$，$y(-1) = 2$

取 Z 变换整理后：

$$
Y(z)(1-bz^{-1}) = by(-1) + X(z)
$$

$$
Y(z) = \frac{by(-1)}{1-bz^{-1}} + \frac{X(z)}{1-bz^{-1}}
$$

代入 $y(-1)=2$，$X(z) = \frac{1}{1-az^{-1}}$：

$$
Y(z) = \underbrace{\frac{2b}{1-bz^{-1}}}_{\text{零输入：只有 } y(-1)} + \underbrace{\frac{1}{(1-az^{-1})(1-bz^{-1})}}_{\text{零状态：只有 } X(z)}
$$

### 验证

- 令 $x(n)=0$（即 $X(z)=0$）：$Y(z) = \frac{2b}{1-bz^{-1}}$ → 只剩零输入响应 ✓
- 令 $y(-1)=0$：$Y(z) = \frac{1}{(1-az^{-1})(1-bz^{-1})}$ → 只剩零状态响应 ✓

## 生活类比

想象一个秋千：
- **零输入响应**：你之前推了一把（初始条件），然后不再推（$x(n)=0$），秋千靠惯性自己摆动 → 慢慢停下来
- **零状态响应**：秋千一开始静止（初始条件=0），你开始有节奏地推（$x(n) \neq 0$），秋千被你推动摆起来
- **完整响应**：秋千之前就在摆（初始条件），你又继续推（输入），最终运动 = 惯性摆动 + 被推动的摆动
