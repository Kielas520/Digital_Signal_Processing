## 第1章 时域离散信号和时域离散系统

**★ 筛选性质**

$$x(n)\delta(n-n_0) = x(n_0)\delta(n-n_0)$$

**★ 周期性判断**

$x(n)=\cos(\omega_0 n)$ 是周期序列 $\iff$ $\frac{\omega_0}{2\pi} = \frac{M}{N}$（有理数，$M,N$ 互质），周期为 $N$

**★ 卷积和**

$$y(n) = x(n)*h(n) = \sum_{m=-\infty}^{+\infty} x(m)h(n-m)$$

卷积结果长度 = $N_1 + N_2 - 1$

**★ 因果性充要条件**

$$h(n) = 0, \quad n < 0$$

**★ 稳定性充要条件**

$$\sum_{n=-\infty}^{+\infty} \lvert h(n)\rvert < +\infty$$

**★ 采样定理**

$$f_s \geq 2f_0 \quad (\Omega_s \geq 2\Omega_c)$$

**数字频率与模拟频率**

$$\omega = \Omega T = 2\pi f / f_s$$

## 第2章 Z变换与傅里叶变换

**★ Z 变换定义**

$$X(z) = \sum_{n=-\infty}^{+\infty} x(n) z^{-n}$$

**★ 常用变换对**

$$a^n u(n) \xleftrightarrow{Z} \frac{1}{1-az^{-1}}, \quad \lvert z\rvert > \lvert a\rvert$$

$$-a^n u(-n-1) \xleftrightarrow{Z} \frac{1}{1-az^{-1}}, \quad \lvert z\rvert < \lvert a\rvert$$

$$\delta(n) \xleftrightarrow{Z} 1$$

$$u(n) \xleftrightarrow{Z} \frac{1}{1-z^{-1}}, \quad \lvert z\rvert > 1$$

**★ 移位性质**

$$x(n-m) \xleftrightarrow{Z} z^{-m}X(z)$$

**★ 卷积定理**

$$x(n)*h(n) \xleftrightarrow{Z} X(z) \cdot H(z)$$

**★ Z 域微分**

$$nx(n) \xleftrightarrow{Z} -z\frac{dX(z)}{dz}$$

**★ DTFT 与 Z 变换关系**

$$X(e^{j\omega}) = X(z)\big|_{z=e^{j\omega}}$$

**★ 系统函数**

$$H(z) = \frac{Y(z)}{X(z)}$$

**★ 因果稳定系统**

$$H(z) \text{ 的所有极点在单位圆内}(\lvert z_k\rvert < 1)$$

**★ 单边Z变换移位（有初始条件）**

$$\mathcal{Z}\{y(n-1)\} = z^{-1}Y(z) + y(-1)$$

## 第3章 离散傅里叶变换 DFT

**★ DFT 正变换**

$$X(k) = \sum_{n=0}^{N-1} x(n) W_N^{kn}, \quad 0 \leq k \leq N-1$$

**★ IDFT 反变换**

$$x(n) = \frac{1}{N}\sum_{k=0}^{N-1} X(k) W_N^{-kn}, \quad 0 \leq n \leq N-1$$

**★ 旋转因子**

$$W_N = e^{-j2\pi/N}$$

**★ 隐含周期性**

$$X(k+N) = X(k), \quad x(n+N) = x(n)$$

**★ 循环卷积与线性卷积相等条件**

$$L \geq N + M - 1$$

**★ 频域采样定理**

$$N \geq M \quad \text{（采样点数 } \geq \text{ 序列长度）}$$

**★ 谱分析参数关系**

$$F = \frac{f_s}{N} = \frac{1}{T_p} = \frac{1}{NT_s}$$

## 第4章 快速傅里叶变换 FFT

**★ 蝶形运算量**

一个蝶形：**1 次复乘 + 2 次复加**

**★ FFT 总运算量**（$N = 2^L$）

$$\text{复数乘法} = \frac{N}{2}\log_2 N, \quad \text{复数加法} = N\log_2 N$$

**★ 直接 DFT 运算量**

$$\text{复数乘法} = N^2$$

**DIT-FFT vs DIF-FFT 顺序**

- DIT：输入**倒序**，输出**正序**，蝶形**先乘后加**
- DIF：输入**正序**，输出**倒序**，蝶形**先加后乘**

**★ IFFT 公式**

$$x(n) = \frac{1}{N}\left[\text{DFT}[X^*(k)]\right]^*$$

## 第5章 网络结构

**★ 直接型画图规则**

- 延迟器个数 = $\max(M, N)$（$H(z)$ 的最高阶次）
- 左边反馈系数 = **分母系数取反**
- 右边前馈系数 = **分子系数保持**

**三种 IIR 结构**

| 结构 | 方法 |
|------|------|
| 直接型 | $H(z)$ → 差分方程 → 直接画 |
| 级联型 | $H(z)$ 因式分解 → 子系统串联 |
| 并联型 | $H(z)$ 部分分式展开 → 子系统并联 |

**三种 FIR 结构**：直接型、级联型、**线性相位型**

**★ 线性相位条件**

$$h(n) = \pm h(N-1-n)$$

## 第6章 IIR 数字滤波器设计

**★ 巴特沃斯幅度平方函数**

$$\lvert H_a(j\Omega)\rvert^2 = \frac{1}{1+(\Omega/\Omega_c)^{2N}}$$

**★ 求阶数 $N$**

$$N \geq \frac{\lg\left(\frac{10^{0.1\alpha_s}-1}{10^{0.1\alpha_p}-1}\right)}{2\lg(\Omega_s/\Omega_p)}$$

**★ 求 3dB 截止频率 $\Omega_c$**

$$\Omega_c = \frac{\Omega_p}{(10^{0.1\alpha_p}-1)^{1/(2N)}}$$

**★ 去归一化**

$$H_a(s) = G_a(p)\big|_{p=s/\Omega_c}$$

**★ 脉冲响应不变法**

$$\frac{A_k}{s-s_k} \to \frac{A_k}{1-e^{s_k T}z^{-1}}$$

频率关系：$\omega = \Omega T$（线性），**有混叠**

**★ 双线性变换法**

$$s = \frac{2}{T} \cdot \frac{1-z^{-1}}{1+z^{-1}}$$

**★ 预畸变公式**

$$\Omega = \frac{2}{T}\tan\left(\frac{\omega}{2}\right)$$

频率关系：非线性，**无混叠**

## 第7章 FIR 数字滤波器设计

**★ 理想低通脉冲响应**

$$h_d(n) = \frac{\sin[\omega_c(n-\alpha)]}{\pi(n-\alpha)}, \quad \alpha = \frac{N-1}{2}$$

$$h_d(\alpha) = \frac{\omega_c}{\pi}$$

**★ 窗函数法**

$$h(n) = h_d(n) \cdot w(n)$$

**★ 海明窗**

$$w(n) = 0.54 - 0.46\cos\left(\frac{2\pi n}{N-1}\right)$$

**常用窗阻带衰减**：矩形窗 $-21$dB，汉宁窗 $-44$dB，海明窗 $-53$dB，布莱克曼窗 $-74$dB

**★ 四种线性相位类型不能实现的滤波器**

- 不能做低通：III 型（奇数反对称）、IV 型（偶数反对称）
- 不能做高通：II 型（偶数正对称）、III 型（奇数反对称）
