# 巴特沃斯多项式查表

## 问题

巴特沃斯滤波器设计中说"查表得 $G_a(p)$"，是根据什么查？怎么查？

## 答案

**根据滤波器阶数 $N$ 查表。** $N$ 算出来是几，就去表里找第几行。

## 巴特沃斯归一化低通原型多项式表

$G_a(p)$ 的形式永远是 $\frac{1}{\text{分母多项式}}$，分子为 1，只需要查分母：

| $N$ | 分母多项式 $B_N(p)$ |
|-----|---------------------|
| 1 | $(p+1)$ |
| 2 | $(p^2 + \sqrt{2}\,p + 1)$，即 $(p^2 + 1.4142p + 1)$ |
| 3 | $(p+1)(p^2 + p + 1)$ |
| 4 | $(p^2 + 0.7654p + 1)(p^2 + 1.8478p + 1)$ |
| 5 | $(p+1)(p^2 + 0.6180p + 1)(p^2 + 1.6180p + 1)$ |
| 6 | $(p^2 + 0.5176p + 1)(p^2 + 1.4142p + 1)(p^2 + 1.9319p + 1)$ |

则：

$$
G_a(p) = \frac{1}{B_N(p)}
$$

## 例子：N=5

$$
G_a(p) = \frac{1}{(p+1)(p^2 + 0.6180p + 1)(p^2 + 1.6180p + 1)}
$$

## 查完之后下一步

将 $p = s / \Omega_c$ 代入 $G_a(p)$，展开化简，就得到实际的模拟滤波器系统函数 $H_a(s)$。

## 考试会怎么考

考试通常会以下三种方式之一给你这个信息：
1. **直接给表**：试卷附录里有巴特沃斯多项式表，你查对应 N 的那行就行
2. **给极点**：给出 N 个归一化极点 $p_k$，让你自己写 $G_a(p)$
3. **直接给 $G_a(p)$**：省掉查表步骤，让你直接去归一化

不会让你现场推导这些系数（系数是数学家提前算好的）。
