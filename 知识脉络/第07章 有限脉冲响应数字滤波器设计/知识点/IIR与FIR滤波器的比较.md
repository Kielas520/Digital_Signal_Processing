# IIR 与 FIR 数字滤波器的比较

## 【通俗理解】

IIR 和 FIR 就像两种不同类型的筛子：IIR 有"记忆"（反馈），效率高但可能不稳定；FIR 没有记忆，安全稳定但更"笨重"（需要更高阶数）。没有绝对的好坏，选哪个取决于具体需求。

---

## 全面对比表

| 对比项 | IIR 滤波器 | FIR 滤波器 |
|--------|-----------|-----------|
| **系统类型** | 递归型（有反馈） | 非递归型（无反馈） |
| **脉冲响应** | 无限长 | 有限长 |
| **系统函数** | $H(z)$ 分子分母都有多项式 | $H(z)$ 只有分子多项式（分母=1） |
| **稳定性** | 不一定稳定（极点可能在单位圆外） | **天生稳定**（无极点，或极点全在 $z=0$） |
| **线性相位** | 一般**无法实现**精确线性相位 | **可以精确实现**线性相位 |
| **设计方法** | 间接法（借助模拟滤波器） | 直接法（窗函数法、频率采样法等） |
| **阶数** | 较低阶就能达到较好的滤波效果 | 通常需要**更高阶数** |
| **计算量** | 同等指标下计算量较小 | 同等指标下计算量较大 |
| **网络结构** | 直接型、级联型、并联型 | 直接型、级联型、线性相位型 |
| **适用场景** | 对相位要求不高的场景（如音频均衡） | 要求线性相位的场景（如数据通信、图像处理） |

---

## 关键记忆点

1. **要线性相位 → 选 FIR**（IIR 做不到精确线性相位）
2. **要低阶高效 → 选 IIR**（同等滤波效果 IIR 阶数更低）
3. **要绝对稳定 → 选 FIR**（无反馈，天生稳定）
4. **设计方法不同**：IIR 借助模拟滤波器间接设计，FIR 直接在数字域设计

---

## 【考卷标答模板】

**题型：简答——IIR 与 FIR 滤波器的主要区别**

> (1) IIR 滤波器的单位脉冲响应无限长，系统有反馈结构，可能不稳定，一般不能实现严格的线性相位，但同等指标下阶数较低、效率较高。
> (2) FIR 滤波器的单位脉冲响应有限长，系统无反馈，天然稳定，可以精确实现线性相位，但同等指标下需要更高的阶数。
> (3) IIR 通常用间接法（借助模拟滤波器）设计，FIR 通常用直接法（窗函数法、频率采样法）设计。
