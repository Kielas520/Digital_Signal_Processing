# 数字信号处理 — 课程全景导览

## 一句话定义

**数字信号处理（Digital Signal Processing, DSP）= 用计算机（数字设备）对信号进行分析和加工。**

你每天都在用它，只是从来没注意过：

- **打电话**：你的声音（模拟信号）被手机采样变成数字，经过降噪、压缩、编码后传到对方手机，再还原成声音。整个过程就是 DSP。
- **听歌**：MP3 文件本身就是一段被压缩过的数字信号，播放时解码、均衡、音效增强全是 DSP。
- **拍照修图**：图像是二维信号，美颜磨皮 = 低通滤波（模糊掉高频噪声和瑕疵），锐化 = 高通滤波（增强边缘细节）。
- **语音助手**：Siri / 小爱同学要先把你说的话做频谱分析（FFT），提取特征，再送进 AI 模型识别。
- **医学影像**：CT、MRI 的原始数据就是一堆频域采样，必须通过逆变换重建出人体截面图像。

所以这门课的本质就是：**教你怎么把现实世界的信号"数字化"，然后用数学工具去分析它、改造它。**

## 各章详细导览

### 第1章 时域离散信号和时域离散系统 —— "把信号变成数字"

**解决的问题**：现实世界的信号（声音、电压波形……）是连续的，但计算机只能处理一个个离散的数字。怎么把连续信号变成离散序列？变完之后还能还原回去吗？

**你要学会的**：
- 认识几种最基本的离散序列：单位脉冲 $\delta(n)$、单位阶跃 $u(n)$、正弦序列、指数序列等
- 序列的基本运算：加、乘、移位、翻转、卷积和
- 线性时不变系统（LTI）的核心性质：线性、时不变、因果性、稳定性
- **采样定理**：采样频率至少是信号最高频率的 2 倍，否则信号就"失真"了（混叠现象）
- 卷积和计算（翻转 → 移位 → 相乘 → 求和）

**生活类比**：就像用一把尺子每隔固定距离在曲线上点一个点。点得够密（采样率够高），曲线就能被这些点完美还原；点得太稀，曲线就被"歪曲"了。

---

### 第2章 时域离散信号和系统的频域分析 —— "换一副眼镜看信号"

**解决的问题**：光看信号的"波形"（时域），很多特征是看不出来的。比如一段音乐里有哪些音高（频率）？这些频率的强度各是多少？我们需要把信号从"时域"变换到"频域"去分析。

**你要学会的**：
- **Z 变换**：把离散序列变成一个关于复变量 $z$ 的函数 $X(z) = \sum_{n} x(n) z^{-n}$，是分析离散系统最强大的数学工具
- Z 变换的收敛域（ROC）：决定了变换是否存在、对应什么类型的序列
- **逆 Z 变换**：从频域变回时域（部分分式展开法、长除法、围线积分法）
- **序列的傅里叶变换（DTFT）**：Z 变换在单位圆上的特例，直接对应信号的"频谱"
- 用 Z 变换解线性常系数差分方程
- 系统函数 $H(z)$ 和频率响应 $H(e^{j\omega})$ 的关系

**生活类比**：时域就像看一段录音的"声波图"——上下抖动的曲线；频域就像看这段录音的"频谱图"——哪些频率强、哪些弱。Z 变换就是帮你在这两种视角之间自由切换的"翻译器"。

---

### 第3章 离散傅里叶变换 DFT —— "让计算机也能算频谱"

**解决的问题**：傅里叶变换的结果是连续的频谱（无穷多个频率点），但计算机只能处理有限个数。怎么把连续频谱"离散化"成有限个点，同时尽量不丢信息？

**你要学会的**：
- **DFT 的定义**：$X(k) = \sum_{n=0}^{N-1} x(n) W_N^{kn}$，其中 $W_N = e^{-j2\pi/N}$
- DFT 隐含的周期性：DFT 本质上是在处理周期信号
- **循环移位**和**循环卷积**（区别于第1章的线性卷积！）
- 循环卷积矩阵的计算方法
- DFT 的共轭对称性
- 频域采样定理：对频谱采样 N 个点，能否唯一还原原始序列？
- 用 DFT 进行谱分析时的注意事项

**生活类比**：DFT 就像是把一首歌的频谱从"高清连续画面"变成"N 个像素的缩略图"。像素够多（N 够大），缩略图就足够清晰。

---

### 第4章 快速傅里叶变换 FFT —— "把慢算法变成快算法"

**解决的问题**：直接算 N 点 DFT 需要 $N^2$ 次复数乘法，当 N 很大时（比如 N=1024），计算量爆炸。有没有更聪明的算法？

**你要学会的**：
- **按时间抽取的 FFT（DIT-FFT）**：把 N 点 DFT 拆成两个 N/2 点 DFT，反复拆下去
- **按频率抽取的 FFT（DIF-FFT）**：另一种拆法，思路类似但蝶形运算结构不同
- 蝶形运算的结构和计算
- 运算量对比：FFT 只需 $\frac{N}{2}\log_2 N$ 次复数乘法（N=1024 时，从 100 万次降到约 5000 次！）
- 用 FFT 计算 IFFT 的技巧

**生活类比**：假设你要给 1024 个人排身高顺序。暴力比较需要上百万次比较；而归并排序（类似 FFT 的分治思想）只需要约 1 万次。FFT 就是频谱计算领域的"归并排序"。

---

### 第5章 时域离散系统的网络结构 —— "把公式变成电路"

**解决的问题**：前面几章都在讨论数学公式。但真正要用硬件或程序实现一个数字系统时，同一个系统函数 $H(z)$ 可以有很多种不同的实现结构，各有优劣。怎么选？

**你要学会的**：
- 信号流图的基本概念
- **IIR 系统的三种基本结构**：直接型、级联型、并联型
- **FIR 系统的结构**：直接型、级联型、线性相位结构、频率采样结构
- 各种结构的优缺点对比（计算量、存储量、系数灵敏度、稳定性）

**生活类比**：同样是做一道菜（实现同一个系统函数），你可以"一锅炖"（直接型），也可以"分步骤流水线"（级联型），还可以"多个灶台并行做再合到一起"（并联型）。味道一样，但效率、可控性不同。

---

### 第6章 IIR 数字滤波器设计 —— "造一个有记忆的筛子"

**解决的问题**：我们经常需要从混合信号中"筛掉"不想要的频率成分（比如去掉电流中的 50Hz 工频干扰）。怎么设计一个数字滤波器来做这件事？IIR（无限脉冲响应）滤波器因为有反馈结构，效率高，用较少的阶数就能实现很陡的滤波特性。

**你要学会的**：
- 滤波器的技术指标：通带截止频率、阻带截止频率、通带最大衰减、阻带最小衰减
- **巴特沃斯（Butterworth）滤波器**的设计方法（最常考）
- **脉冲响应不变法**：把模拟滤波器"搬"到数字域（优点：时域逼近好；缺点：有频率混叠）
- **双线性变换法**：用非线性映射避免混叠（优点：无混叠；缺点：频率关系非线性，需要预畸变）
- 频率变换：从低通原型变换到高通、带通、带阻滤波器

**生活类比**：IIR 滤波器像一个"有记忆的筛子"——它不仅看当前输入，还参考之前的输出（反馈），所以能用很少的参数实现很强的滤波效果，但搞不好会"震荡"（不稳定）。

---

### 第7章 FIR 数字滤波器设计 —— "造一个无记忆的筛子"

**解决的问题**：IIR 滤波器虽然高效，但它的相位响应是非线性的（会让信号波形变形），而且存在不稳定风险。有些场景（比如图像处理、数据通信）要求**严格的线性相位**，这时候就需要 FIR 滤波器。

**你要学会的**：
- **线性相位 FIR 滤波器的四种类型**及其适用的滤波器类型（低通/高通/带通/带阻）
- **窗函数法**：先求理想滤波器的无限长脉冲响应，再用窗函数截断（矩形窗、汉宁窗、海明窗、布莱克曼窗……）
- **频率采样法**：在频域直接指定滤波器响应，再用 IDFT 变换到时域
- 等波纹最佳逼近法（了解即可）
- **IIR 与 FIR 滤波器的全面对比**

**生活类比**：FIR 滤波器像一个"无记忆的筛子"——它只看当前和过去有限个输入，没有反馈，天生稳定，而且可以实现完美的线性相位（信号"不变形"），但代价是需要更多的参数（更高的阶数）。

## 章节依赖关系图

```
第1章（信号与系统基础）
  │
  ├──→ 第2章（Z变换/傅里叶变换 = 频域分析工具）
  │       │
  │       ├──→ 第3章（DFT = 让计算机能算的离散频域）
  │       │       │
  │       │       └──→ 第4章（FFT = DFT的加速算法）
  │       │
  │       ├──→ 第5章（网络结构 = 系统函数的硬件/软件实现）
  │       │
  │       ├──→ 第6章（IIR滤波器设计 = 用模拟滤波器理论设计数字滤波器）
  │       │
  │       └──→ 第7章（FIR滤波器设计 = 直接在数字域设计线性相位滤波器）
```

**第1、2章是地基**，后面所有章节都建立在它们之上。第3、4章是一条线（频域计算），第5、6、7章是另一条线（系统实现与滤波器设计）。

## 考试重点速览

根据课程复习资料，**综合大题的重灾区**是：

1. **Z 变换与逆 Z 变换**（第2章）—— 必考，经常出综合计算题
2. **DFT 与循环卷积**（第3章）—— 矩阵计算、循环卷积与线性卷积的关系
3. **FFT 运算量计算**（第4章）—— 蝶形运算次数、与直接法的对比
4. **网络结构画图**（第5章）—— 给系统函数画直接型/级联型/并联型结构
5. **巴特沃斯滤波器设计**（第6章）—— 完整设计流程（求阶数 → 求截止频率 → 查表 → 去归一化 → 变换到数字域）
6. **IIR 与 FIR 对比**（第6、7章）—— 简答高频考点

选择题和填空题则覆盖所有章节的基本概念。
